# The Anisotropic Shader

Similar to Glossy shader.

Set anisotropy \(how stretched shiny is\) to 0 to match glossy.

anisotropy set to -1 is for CDs or pans.

Can do carbon fiber material.

Doesn't work in EEVEE.
